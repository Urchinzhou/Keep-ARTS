# [88. Merge Sorted Array【Easy】](https://github.com/Urchinzhou/Keep-ARTS/issues/13)

**双指针**
```C
void merge(int* nums1, int nums1Size, int m, int* nums2, int nums2Size, int n){
    int p1 = 0,p2 = 0;
    int sort[m+n];
    int cur = 0;
    while(p1 < m || p2 < n){
        if(p1 == m){
            cur = nums2[p2++];
        }else if(p2 == n){
            cur = nums1[p1++];
        }else if(nums1[p1] <= nums2[p2]){
            cur = nums1[p1++];
        }else{
            cur = nums2[p2++];
        }
        sort[p1 + p2 - 1] = cur;
    }
    for(int i = 0;i < m+n; i++){
        nums1[i] = sort[i];
    }
}
```
**逆向双指针**，相对于双指针，省一个中间变量。
```C
void merge(int* nums1, int nums1Size, int m, int* nums2, int nums2Size, int n){
    int p1 = m -1, p2 = n - 1;
    int sort[m+n];
    while(p1 >= 0 || p2 >= 0){
        if(p1 == -1){
            nums1[p2 + 1] = nums2[p2--];
        }else if(p2 == -1){
            nums1[p1 + 1] = nums1[p1--];
        }else if(nums1[p1] > nums2[p2]){
            nums1[p1 + p2 + 1] = nums1[p1];
            p1 -= 1;
        }else{
            nums1[p1 + p2 + 1] = nums2[p2];
            p2 -= 1;
        }
    }
}
```
