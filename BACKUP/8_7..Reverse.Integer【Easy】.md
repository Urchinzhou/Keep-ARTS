# [7. Reverse Integer【Easy】](https://github.com/Urchinzhou/Keep-ARTS/issues/8)

```C
int reverse(int x){
    int rev = 0;
    while(x != 0){
        int pop = x % 10;
        x /= 10;
        if((rev > INT_MAX / 10) || (rev == INT_MAX / 10 && pop > 7))
            return 0;
        else if((rev < INT_MIN / 10) || (rev == INT_MIN / 10 && pop < -8))
            return 0;
        rev = rev * 10 + pop;
    }
    return rev;
}
```
思路：弹出原始值的低位，压入反转值的高位。
溢出点：```rev * 10 + pop```
